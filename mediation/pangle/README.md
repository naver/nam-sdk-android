# Pangle Mediation for NAM SDK for Android

## Supported ad formats

| Ad format     | Description |
|:--------------|:------------|
| Banner        | O           |
| Native        | O           |
| Native Simple | X           |
| Interstitial  | O           |
| Rewarded      | O           |

## Integration

```gradle
...
repositories {
    maven {
        url "https://artifact.bytedance.com/repository/pangle"
    }
}

dependencies {
    implementation 'com.naver.gfpsdk.mediation:nam-pangle:<latest-version>'  
}
```
